CLI messenger

Terminal Messenger is a simple CLI messenger written in Go. It runs over TCP and keeps the setup easy to launch while still supporting end-to-end encrypted messaging.

The project uses AES-256-GCM for message encryption, a typed JSONL protocol, reconnect support, delivery acknowledgements, and encrypted local chat history.

Stack

Go + TCP

The messenger is written in Go and keeps the networking layer simple with a TCP-based terminal workflow.

Encryption

AES-256-GCM

Messages are end-to-end encrypted, so the transport stays readable for machines but not for anyone outside the conversation.

Protocol

Typed JSONL

The transport format is a typed JSONL protocol, which keeps the data flow explicit and easy to inspect during development.

Reliability

Reconnect + acknowledgements

The app supports reconnect logic, delivery acknowledgements, and encrypted local history for a more usable terminal chat experience.
