CLI telegram

CLI_Telegram is Telegram in the terminal. I built it because I spent a long time trying to find a working terminal Telegram client, but most of the existing ones are no longer maintained and already do not work properly.

One of the features I really wanted was direct image rendering in the terminal, so the client supports ASCII art output for images right inside the CLI workflow.

Open GitHub repo

Why I built it

Terminal clients were outdated

The project started from a practical need: existing CLI Telegram clients were either abandoned, broken, or too outdated to be useful anymore.

Interface

Telegram in the terminal

The goal is a Telegram experience that stays inside the terminal instead of forcing a switch to a heavy desktop client.

Image rendering

ASCII art in terminal

Images can be rendered directly in the terminal as ASCII art, which makes media support much more interesting than plain text-only chat.

Project direction

Simple and usable

The focus is not on cloning every Telegram feature, but on keeping the terminal client useful, readable, and actually runnable today.
