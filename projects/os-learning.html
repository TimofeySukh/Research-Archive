OS_learning

I wanted to write my own kernel from scratch, so I started building a learning plan around operating systems, low-level architecture, and the path required to get there.

Right now the project is frozen because other topics became more interesting to me, but the repository still reflects that original systems direction and study roadmap.

Open GitHub repo

Goal

Write a kernel from scratch

The core ambition was to move beyond using systems and start understanding how to build one from the lowest layers upward.

Approach

Structured learning plan

Instead of jumping straight into code, the project focused on creating a roadmap for learning the theory and technical pieces behind operating systems.

Topic area

Low-level systems and architecture

The work sits in the space between OS internals, boot processes, memory concepts, and the engineering mindset required for kernel-level work.

Current state

Frozen for now

The project is paused because my focus shifted to other areas, but it remains part of the archive as a serious direction I explored.
