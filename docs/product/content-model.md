# Content Model

## Year groups

The archive is grouped by year, ordered from newest to oldest.

## Project records

Each project record includes:

- `slug`: Used for the project page filename.
- `title`: The visible title on the homepage and project page.
- `description`: A short supporting line for the timeline card.
