# AGENTS

## Working rules

- Keep all user-facing text, code comments, and documentation in English.
- Commit changes only when requested.
- Do not push to GitHub unless the user explicitly asks for it.
- Update `docs/` and `README.md` after every meaningful change.
- Keep project data centralized so new years and projects are easy to add.
